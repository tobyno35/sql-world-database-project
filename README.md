
    # SQL World Database Project
    This project contains SQL queries performed on a "world" database as part of a data analysis lab.
    ## Key Learning Objectives:
    - Filtering records using the WHERE clause.
    - Using ranges with the BETWEEN operator.
    - Pattern matching with the LIKE operator and wildcards.
    - Using aggregate functions like SUM().
    - Implementing column aliases (AS) and string functions (LOWER()).
